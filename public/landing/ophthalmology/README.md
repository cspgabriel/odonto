# Ophthalmology Landing Page Assets

Replace these files to customize your ophthalmology clinic's landing page.

## Branding

| File | Purpose |
|------|---------|
| logo.svg | Light mode logo (navbar, footer) |
| logo-dark.svg | Dark mode logo |
| favicon.svg | Browser tab icon |

## How to customize

1. Replace any file with your own image (keep the same filename)
2. Or update paths in: **Dashboard → Landing Settings → Branding**
3. Recommended formats: SVG for logos
