# Dental Landing Page Assets

Replace these files to customize your dental clinic's landing page.

## Branding

| File | Purpose |
|------|---------|
| logo.svg | Light mode logo (navbar, footer) |
| logo-dark.svg | Dark mode logo |
| favicon.svg | Browser tab icon |

## Images

| Folder | Purpose |
|--------|---------|
| images/ | Hero avatars, doctor photos, service icons, smile comparison, etc. |

Key images used by components:
- `avatar1-8.*.webp` — Hero and doctor avatars
- `img1-6.*.webp` — Hero, we-care, decorative images
- `dr.png` — About Doctor profile
- `before1.*.webp`, `after1.*.webp` — Smile comparison
- `braces.png`, `canal.png` — Service icons

## How to customize

1. Replace any file with your own image (keep the same filename for minimal config changes)
2. Or update paths in: **Dashboard → Landing Settings → Branding**
3. Recommended formats: WebP for photos, SVG for logos

## Note

Tooth chart images are in `/public/teeth/` — do not move them. They are used by the odontogram component.
