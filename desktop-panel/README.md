# Odonto Painel Desktop

App Windows para abrir apenas o painel em modo desktop.

## Comandos

```powershell
npm install
npm run dist
```

Saida:

```text
desktop-panel/dist/Odonto-Painel-1.0.0.exe
```

URL padrao:

```text
https://odonto-gules.vercel.app/dashboard
```

Para trocar:

```powershell
$env:ODONTO_PANEL_URL="https://seu-dominio/dashboard"; npm start
```
